# 05 - Implement

Objetivo: executar apenas a task atual dentro do escopo declarado.

## Regras

1. Leia `specRefs` e `context.readOnly` antes de editar.
2. Preserve mudancas nao relacionadas do usuario.
3. Nao faca refatoracao ampla fora do escopo.
4. Mantenha o core neutro e o adaptador isolado.

## Saida

- Codigo, docs ou templates alterados apenas no escopo declarado.
- Nenhuma funcionalidade de medio ou longo prazo ativada por acidente.
