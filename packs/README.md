# Packs

Status: planned

Packs are optional domain bundles. They can group planned agents, skills and rules for a domain, but are not loaded automatically into the AgentOS core.

## Available Packs

- `engineering`
- `business`
- `isp`
- `devops`
- `security`

## Future Activation

A future project setup task may select a pack, copy or reference its material, then verify the selected files. Until then, packs are documented placeholders.
