# Claude Code Adapter

Status: experimental

This is a future adapter placeholder for Claude Code. It is not complete in this phase and must not be used as the primary AgentOS path.

## Purpose

The adapter may eventually map AgentOS core material into Claude Code-compatible files such as `.claude/agents`, `.claude/skills` and `CLAUDE.md`.

## Boundaries

- AgentOS core does not depend on Claude Code.
- This adapter does not create `.claude/` automatically.
- Codex remains the initial supported adapter.
- Planned mappings are documented in `mapping.yaml`.
