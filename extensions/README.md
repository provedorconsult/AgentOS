# Extensions

Status: planned

Extensions are optional future capabilities. They are outside `core/` and are not loaded automatically.

## Planned Areas

- `memory`: markdown-first memory.
- `heartbeats`: proactive health checks.
- `routines`: future scheduled routines.
- `channels`: future communication interfaces.
- `knowledge-db`: future database-backed knowledge layer.

## Boundary

No scheduler, database, dashboard or integration is implemented in this phase.
